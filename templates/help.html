Центр Допомоги


Як шукати на сайті?
  1. На будь-якій сторінці сайту знайдіть на верхній панелі поле для пошуку
  2. Введіть теги за якими бажаєте шукати (між тегами - ..)
  3. Натисніть Search

Як зареєструватися?
  1. Перейдіть на сторінку
  2. Заповніть усі поля та пройдіть ReCaptcha
  3. Натисніть зареєструватися

Як змінити пароль, нікнейм, ел. пошту?
  1. Перейдіть на сторінку
  2. Якщо ви не авторизовані, то ведіть ім'я користувача або електронну адресу у поле й натисніть відправити листа. Інакше, просто натисніть цю кнопку.
  3. Знайдіть код на пошті, яку вказували при реєстрації
  4. Поверніться на сайт і якщо ви не авторизовані, то знову введіть ті самі дані, але у поле нижче(якщо ви авторизовані, то можете зімнити також свій нікнейм або ел. пошту)
  5. Введіть пароль, якщо хочете його змінити та код для підтвердження оновлення
  6. Пройдіть ReCaptcha та натисніть підтвердити

Як підтвердити ел. адресу?
  1. Перейдіть на сторінку
  2. Натисніть відправити листа
  3. Введіть код у поле
  4. Пройдіть ReCaptcha та натисніть підтвердити

Як додати аватар профілю?
  1. Перейдіть до свого аккаунту
  2. Натисніть Огляд...
  3. Виберіть фотографію для завантаження
  4. Натисніть завантажити

Як додати статтю/нотатку?
  1. На верхній панелі натисніть Create
  2. Заповніть поля: назва, опис, текст, теги(опціонально) та обкладинка(якщо є потрібний план)
  3. Виберіть, чи буде ваша нотатка публічною, чи ні
  4. Пройдіть ReCaptcha та натисніть створити

Як змінити статтю/нотатку?
  1. Перейдіть у свій аккаунт та виберіть потрібну статтю/нотатку
  2. Натисніть змінити
  3. Змініть ті поля, які потрібно змінити, ті, що не потрібно змінювати залиште порожніми
  4. Пройдіть ReCaptcha та натисніть змінити

Як видалити статтю/нотатку?
  1. Перейдіть у свій аккаунт та виберіть потрібну статтю/нотатку
  2. Натисніть видалити

Де я можу подивитися обрані статті? На верхній панелі виберіть сторінку Favourites

Які статті потрапляють на сторінку For You? На сторінку For You потрапляють статті з ваших підписок

Чи можуть інші користувачі бачити ваші репости? Так, ваші репости можуть бачити абсолютно усі користувачі.

Навіщо потрібна Page(гарна сторінка акаунту)? Цю сторінку ви можете використовувати для опису профілю, перенаправлення на інші сайти через неї або просто, щоб створити дуже гарну сторінку аккаунту

Як створити Page(гарна сторінка акаунту)? Перейдіть на сторінку та створіть свій HTML код для сторнки
